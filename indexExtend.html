IncludHtml.js

Эта маленькая библиотека предназначена
для динамической вставки в Html, частей других Html файлов

[index.html]    indexExtend.html

сщдержимое элемента id="extId1"
из внешнего файла "./inc/block1.inc.html":

    

    

      ---------------- -----------------
    

    
    {
      "onLoadCalback": "inc_Onload",        // вызываетсся при вставке 
                                            // каждого внешнего элемента
      "incFile": "./inc/block1.inc.html",   // если указан - URL внешнего файла
                                            // иначе используется текущий документ
      "incFromId": "extId1"                  // id вставляемого элемента
      "incInner": false,  // incInner:false, (по умолчению) вставляется outerHTML 
                          // иначе innerHtml
    }
